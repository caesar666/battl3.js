battl3.js
=========

A algorithm battl3 based on javascript for developers.

Setup Project on your computer (Linux links)
=========

Projects dependencies
	* Node.js
		* Express
		* EJS
		* Mongojs
	* Mondodb
	* git


The collections on Mongodb are created automatically.
